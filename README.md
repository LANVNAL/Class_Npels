# 使用教程
* 1.将这个脚本下载下来！
* 2.打开在程序文件夹目录打开cmd界面　输入命令 `pip install -r requirements.txt` 等待安装完成
* 3.打开在程序文件夹目录打开cmd界面 输入命令 `python run.py` 按提示输入你的账号密码
